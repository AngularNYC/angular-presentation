What is Visual Studio Code?

Visual Studio Code is a free open source code editor developed by Microsoft

  • Built with Github Electron, utilize Web Technology (HTML, CSS, Javascript)
  • Offers a robust extensible architecture and highly customizable
  • Editor has Built in...
    • Source Control (Ctrl+Shift+G)
    • Extensions Markert (Ctrl+Shift+X)
    • Debugger (Ctrl+Shift+D)
    • Terminal (Ctrl+~)

Common keyboard shortcut

  • Formats your code (Shift+Alt+F)
  • Open New Window (Ctrl+Shift+N)
  • Open Project Folder (Ctrl+K,Ctrl+O)
  • Go to File (Ctrl+P)
  • Expand Selection (Shift+Alt+Right)
  • Search multiple File (Ctrl+Shift+F)
  • Go to Definition (F12)
  • Rename (F2)
  • Multiple Cursor(Alt+Mouse)
