Templates / Interpolation / Properties

When giving a value to the element's attribute, you can use String Interpolation {{ curlies }}

Templates / Property Binding

Another option is to use property binding [attribute] = property

By doing property binding you do not need "{{ curlies }}" to specify your property. Also, while brackets are used for attribute binding: [attribute], "bind-" can also be used: bind-attribute.

Templates / Data binding extras

Angular supports more advanced bindings than just attribute name

Templates / Event binding: (event)

The bindings directives you've met so far flow data in one direction: from a component to an element

While Parentheses are used for event binding: (event), "on-" can also be used: on-event.

Templates / Reference binding ( #var )

Sometimes we need to access HTML element or Angular component from the template. We can mark it with #userName, and it becomes available as userName everywhere in the template

Templates / Conditional Display (*ngIf)

This conditional expression will add or remove an element from the DOM if it evaluates as a truthy.
