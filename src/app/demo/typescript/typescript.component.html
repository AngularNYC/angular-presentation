TypeScript

TypeScript is a language that compiles to JavaScript

  • Created by Anders Hejlsberg, lead architect of Turbo Pascal, Delphi and later C#
  • Rather than try to replace JavaScript, TypeScript embraces and extends it.
  • TypeScript adds an optional Type System as well as features from the latest versions of JavaScript not yet supported in all browsers otherwise

TypeSystem

2 + 2 = 4? Not always the case in JavaScript. Luckily typescript allows us to add types which allows to detect the issue early.

TypeScript has different types for different units of data. These types are:

  • Boolean
  • Number
  • String
  • Array
  • Tuple
  • Enum
  • Any
  • Void
  • Null/Undefined
  • Never

Classes

TypeScript has classes, and Angular makes heavy use of them.

They work similar to classes in other languages, and serve for grouping methods and properties belonging to one object together.

Classes - constructor

There's a special method on the class called constructor. It's run when the class is created and allows the class to take parameters

Classes - export

By the way, did you notice export before class? This is used to share information between files. We're going to import and use this in the next slide (for higher effect)

Classes - import

Imports and exports are not just for classes. They work with variables, functions and other things!

TypeScript supports lots of other cool features such as:

  • Interfaces
  • Enums
  • Async/await
  • Getters/setters
  • Desctructuring
  • Arrow functions
  • And more!

We won't cover them in detail, check out the TypeScript website!
